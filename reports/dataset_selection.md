# Dataset Selection

## Project

ML Prediction Service

## Selected Dataset

Telco Customer Churn

## Dataset Source

Kaggle - Telco Customer Churn by BlastChar

## Dataset Size

The dataset contains 7,043 customer records and 21 columns.

## Problem Type

Binary classification

## Target Variable

`Churn`

## Reason for Selection

This dataset was selected because it is suitable for building a production-style machine learning prediction service. It has a clear business problem, a clear target variable, and a mix of categorical and numerical customer features.

The dataset is small enough for fast iteration but rich enough to practice preprocessing, model training, evaluation, artifact saving, and API-based prediction.

## Expected Input Features

The model may use customer-related features such as:

- Tenure
- Contract type
- Monthly charges
- Total charges
- Internet service
- Payment method
- Online security
- Tech support
- Paperless billing

## Primary Metric

F1-score

## Secondary Metrics

- Accuracy
- Precision
- Recall
- ROC-AUC

## Limitation

This is a sample/fictional telco dataset, so the project should be presented as an ML engineering practice project rather than a real telecom business analysis.
